Help

Данное веб-приложение используется для нескольких видов операций: поиск, определение языка, реферирование. Первая операция - это информационно-поисковая система со стратегией логического поиска. Она позволяет находить документы по поиску через слова, находящиеся в этих документах. Существует два вида поиска: поиск по точно заданному слову и поиск по релевантности. Сам поиск осуществляется через ввод в форму слов, которые должны быть в желаемом документе и отправки этой формы по нажатию клавиши "Enter" или по нажатию кнопки "Найти". Весь список документов появляется на этой же странице. Сам список состоит из ссылки, по которой можно перейти на страницу самого документа, т.е. с полным его описанием, и краткой информации о релевантности и найденных терминах. Для поиска по релевантности список документов сортируется по возрастанию релевантности. Как дополнительный функционал для поиска по релевантности реализованы поиск по синонимам, исправление орфографических ошибок и поиск не учитывающий число слова. Также при поиске вычисляются основные метрики, позволяющие оценить качество работы информационно-поисковой системы.
Вторая операция - это определение языка. Язык определяется между французским и английским языками (другие не поддерживаются). Для определения языка пользователью предоставлен выбор метода из трёх вариантов: метод N-грамм, алфавитный метод и нейросетевой метод. Выбрав метод пользователь должен ввести абсолютный путь к html файлу в поле ввода и отправить форму, нажав на кнопку "Check". После отправки формы, пользователь увидит результат определения языка и информацию о том, сколько времени заняла обработка.
Также пользователю будет предоставлена возможность сохранить результат. Третья операция - это реферирование. Пользователь предоставлен выбор из двух методов, а именно "Sentence extraction" и "Machine Learning". Также пользователь может получить ключевые слова текста, отметив галочкой поле "Keywords". После выбора метода пользователь должен либо ввести имя уже существующего документа, либо загрузить документ из файла и отправить форму. Отправив форму пользователь увидит информацию о документе, содержащию в себе несколько строк реферата и ключевых слов(если они были выбраны), ссылку на документ и возможность сохранения реферата. При переходе по ссылке документа, пользователь попадёт на страницу документа, где увидит полный список ключевых слов(если они были выбраны), весь реферат и оригинальный текст.
