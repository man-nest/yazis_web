Help

Данное веб-приложение является информационно-поисковой системой со стратегией логического поиска.
Оно позволяет находить документы по поиску через слова, находящиеся в этих документах.

Существует два вида поиска: поиск по точно заданному слову и поиск по релевантности. Сам поиск осуществляется через ввод в форму слов, которые должны быть в желаемом документе и отправки этой формы по нажатию клавиши "Enter" или по нажатию кнопки "Найти".
Весь список документов появляется на этой же странице.

Сам список состоит из ссылки, по которой можно перейти на страницу самого документа, т.е. с полным его описанием, и краткой информации о релевантности и найденных терминах.
Для поиска по релевантности список документов сортируется по возрастанию релевантности. Как дополнительный функционал для поиска по релевантности реализованы поиск по синонимам, исправление орфографических ошибок и поиск не учитывающий число слова.

Также при поиске вычисляются основные метрики, позволяющие оценить качество работы информационно-поисковой системы.
